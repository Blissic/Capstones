Network Fundamentals

Essential concepts in computer networking

Network Architecture

Modern networks use layered architectures with distinct components. The OSI model has seven layers, while TCP/IP uses four layers. These models aid troubleshooting and system design.

Transmission Media

Networks use twisted pair cables, fiber optics, and wireless frequencies. Each medium affects bandwidth, latency, and deployment. Fiber optics excel in long-distance connections.

Network Protocols

Protocols define device communication rules. Key protocols include TCP/IP for reliability, UDP for low latency, and HTTP/HTTPS for web traffic. IPv6 adoption is increasing.
